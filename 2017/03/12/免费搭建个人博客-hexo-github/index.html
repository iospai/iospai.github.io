#Markdown 语法说明 (简体中文版)

说明：

1、MarkDown和HTML二者兼容，不需额外标注；

2、制约，HTML区块元素——比如：<div>、<table>、<pre>、<p>等标签，必须在前后加空行与其他内容隔开，并且其开始标签和结束标签不能使用制表符或空格来缩进；

3、在HTML区块标签间的Markdown格式语法将不会被处理。例如：

# 一级标题

4、特殊字符处理。<和&，分别对应&lt;和&amp;
5、版权符号 © 对应 &copy;
6、强制换行，<br/>或者在插入处键入两个以上的空格然后回车。

- (void)sendLog:(CIInvokedCommand *)command {
NSString *content = [command argStringAtIndex:0];
NSString *level = [command argStringAtIndex:1];//打印的log信息 "i","v","d","w","e"
if ([level be_isValidString]) {
    if ([@"E" isEqualToString:level] || [@"e" isEqualToString:level]) {
        DDLogError(content);
    } else if ([@"W" isEqualToString:level] || [@"w" isEqualToString:level]) {
        DDLogWarn(content);                            // Orange
    } else if ([@"D" isEqualToString:level] || [@"d" isEqualToString:level]) {
        DDLogDebug(content);
    } else if ([@"V" isEqualToString:level] || [@"v" isEqualToString:level]) {
        DDLogVerbose(content);              // Default (black)
    } else {
        DDLogInfo(content);  // Default (black)
    }
} else {
    DDLogInfo(content);
}
}
