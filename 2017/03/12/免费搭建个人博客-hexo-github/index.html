免费搭建个人博客 hexo+github

之前尝试过各种方式的个人博客。
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown 语法编辑文章，在几秒内，即可利用靓丽的主题生成静态网页。

问题列表

1、Deployer not found: git

npm install hexo-deployer-git --save

2、hexo部署成功后，域名访问GitHub Page绑定的域名地址，报404错误页面。

由于CNAME被删除，需要把CNAME文件放在source文件夹，即可。
cd source
vi CNAME
写入绑定的域名，保存重新发布即可。
